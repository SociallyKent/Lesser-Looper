### Lesser-Looper

Controls:

#### To Start looping

1. Single Click
- Loop From song timing line

2. Double Click
- Loop From start (First)
- Loop Remove (Second)

#### To Move window

Hold and drag Right Click on Button

### Lesser-Looper
操作方法

#### ループの開始
シングルクリック
- 曲のタイミングラインからループを開始します。

ダブルクリック
- 1回目：曲の先頭からループを開始します。
- 2回目：ループを解除します。

#### ウィンドウの移動

ボタン上で右クリックを押したままドラッグすると。
